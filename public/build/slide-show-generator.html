Slide Show Project

Here's a high level overview of how the slide show generator works. The project is hosted on Github.

  • The goal of the project was to make it easy to create static site slide shows which could be hosted on Netlify, S3, etc.
  • To add a slide run node add-slide.js slide-name.html "slide title" in the root of the project. This will stub out a slide template, and append it to the list of slides in slides.json.
  • Running npm run build will loop through the templates in slides.json, and using Webpack, will generate the footer partial, and all slide html templates.
  • Generated HTML, Javascript, and CSS files get written to public/build.
  • Client side, the Back and Next buttons, as well as the select list, make ajax requests to retrieve slide templates, and swaps in only the content portion of the slide. The footer is persistent across slide navigations.
  • Stimulus was used to wire up actions to the interactive elements on the page, as well as to keep the browser history up to date.
